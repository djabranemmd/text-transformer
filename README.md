# 🔄 Text Transformer | حوّلها لي

A simple and interactive web application that allows users to transform text instantly using different operations.

تطبيق ويب بسيط وتفاعلي يسمح للمستخدم بتحويل النصوص بسرعة باستخدام عمليات مختلفة.

---

## 🎯 Project Idea | فكرة المشروع

This project helps users manipulate text easily without writing code.

يساعد هذا المشروع المستخدم على تعديل النصوص بسهولة بدون الحاجة لكتابة أكواد.

It provides quick tools for:
يوفر أدوات سريعة لـ:

- Uppercase conversion (تحويل إلى حروف كبيرة)
- Lowercase conversion (تحويل إلى حروف صغيرة)
- Reverse text (عكس النص)
- Remove spaces (حذف الفراغات)

---

## ⚙️ How It Works | كيف يعمل

1. User enters a text input  
   المستخدم يكتب نص في الخانة

2. Chooses an operation  
   يختار العملية المطلوبة

3. Result is displayed instantly  
   تظهر النتيجة مباشرة
